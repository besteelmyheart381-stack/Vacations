# 🧳 Warren Family Vacation Planner — PWA

A Progressive Web App you can install directly on your iPhone or Android home screen.

---

## 🚀 Deploy to GitHub Pages (Step-by-Step)

### Step 1 — Create a GitHub Account
If you don't have one: go to **github.com** → Sign Up (it's free).

---

### Step 2 — Create a New Repository
1. Go to **github.com** and sign in
2. Click the **+** in the top-right corner → **New repository**
3. Name it: `vacations` (or anything you like)
4. Set it to **Public**
5. Click **Create repository**

---

### Step 3 — Upload Your Files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop ALL of these files/folders:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/` folder (with both icon files inside)
3. Click **Commit changes**

---

### Step 4 — Enable GitHub Pages
1. In your repo, go to **Settings** (top tab)
2. Click **Pages** in the left sidebar
3. Under "Branch", select **main** → **/ (root)**
4. Click **Save**
5. Wait ~60 seconds, then your site is live at:
   `https://YOUR-USERNAME.github.io/vacations/`

---

### Step 5 — Add to iPhone Home Screen
1. Open **Safari** on your iPhone (must be Safari, not Chrome)
2. Go to your GitHub Pages URL
3. Tap the **Share** button (box with arrow) at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **Add** — done! 🎉

The app icon will appear on your home screen just like a native app.

---

### Step 6 — Add to Android Home Screen
1. Open **Chrome** on your Android phone
2. Go to your GitHub Pages URL
3. A banner may appear asking to install — tap **Install**
4. Or tap the **⋮ menu** → **Add to Home screen**

---

## 📱 Features
- Works fully **offline** after first visit
- All data saved locally on your device
- Export/Import your data as a backup file
- Full-screen app experience (no browser UI)

## 💾 Your Data
Everything is stored on **your device** using localStorage.  
Use **Menu → Export Data** to back up your trip info anytime.

---

## 🔄 Updating the App
To update the app after making changes:
1. Edit your files
2. Go to your GitHub repo
3. Upload the updated files (they'll replace the old ones)
4. GitHub Pages updates automatically within a minute
